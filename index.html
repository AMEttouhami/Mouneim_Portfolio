Analyzing Flight Traffic Data using Python and Pandas

The Flight Delay dataset lists flight data for 14 major U.S. airlines for the year 2015. It consists of three tables: Flights, Airports and Airlines. We do a little bit of data cleaning, and then we explore the relationships between these three tables and try to answer basic questions like identifying the flights and airports with the most flight delays, and examine the effects of seasonality, etc., using Python and the Pandas library.

Analyzing Soccer Data With SQL

The European Soccer Database is a database of soccer data about 25,000 matches, 300 teams, and 10,000 players from 11 European countries, covering various teams’ and players’ statistics during the period from 2008 to 2016. It has 7 tables in total: Country, League, Match, Player, Player_Attributes, Team and Team_Attributes. In this project we have used SQLite to analyze these tables, starting with very simple queries and gradually increasing the level of difficulty to more complex ones.

Regression: Predicting Used Car Prices

We build a linear regression model using scikit-learn to predict used car prices. By carefully cleaning the data and removing the outliers in the training set, we show that a linear regression model with polynomial features is quite successful at predicting the price of used cars with good accuracy, with a coefficient of determination R^2 close to 0.94 on the test set. Using the linear regression model from this project, we used Streamlit to build a client-facing web app to predict used car prices.
