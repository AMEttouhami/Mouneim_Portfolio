Regression: Predicting Used Car Prices

April 4, 2024

We build a linear regression model using scikit-learn to predict used car prices. By carefully cleaning the data and removing the outliers in the training set, we show that a linear regression model with polynomial features is quite successful at predicting the price of used cars with good accuracy, with a coefficient of determination R^2 close to 0.94 on the test set.

Using the linear regression model from this project, we used Streamlit to build a client-facing web app to predict used car prices.
