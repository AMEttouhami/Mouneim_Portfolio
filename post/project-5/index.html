Classification: Using Machine Learning to Detect Credit Card Fraud

April 2, 2024

In this project, we tackle the difficult problem of classification on a heavily imbalanced dataset. We try to mitigate the imbalanced nature of the data by using different strategies, like using balanced class weights, and using the so-called synthetic minority oversampling technique (SMOTE, for short). Using these techniques we are able to achieve values of the precision and recall that are in the range of 82% on the test set.
