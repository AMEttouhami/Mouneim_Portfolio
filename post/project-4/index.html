Classification: Twitter Sentiment Analysis

April 3, 2024

In this project, we address the classic problem of assigning either a positive or a negative sentiment to a given statement by using Natural Language Processing (NLP) techniques to learn how to infer sentiments from data. Here, we used the Sentiment140 dataset from Kaggle which has over 1.6 million labelled tweets. Using BERT and DistilBERT models, we were able to achieve good classification performance, with values of the accuracy, precision and recall all in the range of 80% on the test set.
